Anchored Notes는 웹 페이지에 고정되는 스티커 메모를 남길 수 있게 해줍니다. 각 메모는 네 가지 범위 중 하나에 고정되며, 해당 범위가 일치하는 곳에서 다시 나타납니다:

• Global — 모든 페이지에 표시
• Site — 한 사이트의 모든 페이지에 표시
• Page — 정확한 URL 하나에 표시
• Tab — 닫힐 때까지 단일 탭을 따라감

완전한 markdown 에디터에서 작성하세요: 제목, 목록, 클릭 가능한 체크박스가 있는 할 일 목록, 표, 인용, 코드 블록, 그리고 Notion 스타일 "/" 메뉴까지 지원합니다. 메모를 드래그하고, 크기를 조절하고, 색상을 바꿀 수 있습니다(7가지 색상). 16개 언어로 제공됩니다.

Google로 로그인하면 여러 기기에서 메모를 동기화할 수 있습니다. 동기화된 메모는 항상 암호화됩니다. 선택 사항인 암호를 설정하면 종단 간 암호화로 업그레이드되어 오직 본인만 읽을 수 있습니다.

무료: 계정 없이 10개 메모, 무료 계정으로 30개 메모. Pro: 무제한 메모.
